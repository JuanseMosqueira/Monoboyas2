openapi: 3.0.3
info:
  title: API Sistema de Monitoreo de Monoboyas
  description: |
    API REST del sistema de monitoreo de operaciones de transferencia de hidrocarburos
    en monoboyas (Single Point Mooring). Expone los endpoints que consume la aplicación
    web del frontend para gestión de usuarios, operaciones, mediciones, alertas y
    configuración de umbrales.

    Esta API NO recibe datos crudos de sensores: la ingestión de telemetría la realiza
    el Motor de Procesamiento, que escribe directo a la base de datos consultando APIs
    externas (Open-Meteo) o archivos mock según el tipo de sensor.
  version: 1.0.0
  contact:
    name: Proyecto Monoboyas - UNS - Arquitectura y Diseño de Sistemas 2026
